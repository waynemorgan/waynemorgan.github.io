Section 2: CSS

The next part is CSS! CSS can be used to style various parts of a webpage!

Let's check out some examples:

Name Description Statement Example
Color Changes the color of the text color: #ff0000; This text is red!
Size Changes the size of the text font-size: 20px; This text is bigger!
Font Changes the font of the text font-family: serif; This text is a different font!
Weight Changes the weight of the text, similar to the bold tag font-weight: 600; This text is a different weight!
Decoration Changes the decoration of the text text-decoration: line-through; This text has a line through it!
